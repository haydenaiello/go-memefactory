# go-memefactory

Program that makes text have w0NkY letters.

## Installation

```bash
go get github.com/haydenaiello/go-memefactory
```

## Usage

```Go
package main

import (
    "fmt"
    "github.com/haydenaiello/go-memefactory"
)

func main() {
    fmt.Println(memefactory.MakeMeme("insert_meme"))
}
```
